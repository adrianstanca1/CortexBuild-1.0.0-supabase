## ⚠️ **IMPORTANT NOTES**

### **Backend Server:**
The Vercel deployment only includes the **frontend** (static files). The backend server is **NOT deployed** to Vercel.

**To use the full application with backend:**
1. Keep your local backend server running: `npm run server`
2. Or deploy the backend separately to a service like:
   - Render
   - Railway
   - Heroku
   - DigitalOcean
   - AWS

### **Database:**
The SQLite database is local and not included in the Vercel deployment. For production:
1. Use a cloud database (PostgreSQL, MySQL, etc.)
2. Update backend to connect to cloud database
3. Deploy backend to a server platform
