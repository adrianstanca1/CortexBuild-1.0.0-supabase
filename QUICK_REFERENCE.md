# ⚡ Quick Reference - Developer Dashboard

## 🚀 **Start Application**

**IMPORTANT**: You need to run BOTH frontend and backend:

```bash
npm run dev:all
```

This starts:

- **Frontend**: <http://localhost:3000/>
- **Backend**: <http://localhost:3001/>

**Alternative** (run separately):

```bash
# Terminal 1 - Frontend
npm run dev

# Terminal 2 - Backend
npm run server
```
